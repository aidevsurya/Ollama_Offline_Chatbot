#Installation
Install Required Tools First:
<b> python 3.10.19 </b>
<b> virtualenv </b>

<h4> Step 1: </h4>
  <b> Run: </b> "virtualenv MyChatbot"
  
<h4> Step 2: </h4>
  Run Terminal and type this command:
  <code> ./setup.sh </code>

<h4> Step 3: </h4>
  Setup Ollama, Install models, Run ollama service

<h4> Step 4: </h4>
  Run in Terminal:
  <code> python chatbot.py </code>
